## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone git@github.com:KaiqueCovo/dashgo.git

# Go into the repository
$ cd dashgo

# Install dependencies
$ yarn install

# Run tests
$ yarn test

# Run the application
$ yarn dev
```
